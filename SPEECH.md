# ピラミッド犬 セリフ一覧

犬が吹き出しで喋る可能性のある全セリフと、その発生条件。
実体は `main.js` の `LINES_IYASHI`(いやしモード)/ `LINES_DOKU`(どくぜつモード)
(タグ→セリフ配列)。1回につき配列からランダムで1つ選ばれる。
複数匹いるときは「喋る犬」に吹き出しが追従する。

## モード(いやし ⇄ どくぜつ)

右上の 😇/😈 ボタンで切り替え。`mode`(`localStorage.pd_mode`、'iyashi' | 'doku')に
応じて `linePool(tag)` が参照するセリフ集合を切り替える。おやつを食べた感想
(`TREATS[kind].lines` / `linesDoku`)も同様に切り替わる。ツンデレ設定(毒舌モードは
口は悪いが根はデレ)。切替の瞬間は下記の固定セリフを1回だけ表示する(`LINES` のタグ
ではなく直接 `showBubble`):

| 切替方向 | セリフ |
|---|---|
| いやし → どくぜつ | ふん、ここからは しんらつに いくからな |
| どくぜつ → いやし | …なんてね。ほんとは だいすきだよ |

## メモ

- 「きみ」は飼い主(あなた)を指す一般的な呼びかけ。名前機能は現状なし。
  もし名前機能を戻すなら、飼い主呼びを名前にするか犬に名付けるかを決めてから
  該当セリフを差し替える(ROADMAP / チャット履歴参照)。
- セリフを足すとき: `main.js` の `LINES_IYASHI` / `LINES_DOKU` に追記 → この表にも1行足す
  (両モードでタグ構成を必ず揃える)。
- 新しい発生タイミングを足すときは、その場所で `speak(dog, 'タグ', 秒)` を呼ぶ
  (内部で `linePool(tag)` を通してモードに応じたセリフ集合が選ばれる)。
